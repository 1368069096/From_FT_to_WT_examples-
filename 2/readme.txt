本程序为知乎文章'从傅里叶变换进阶到小波变换（二）,https://zhuanlan.zhihu.com/p/66246381'的源代码。
1、'left_right_FFT.m'程序计算了'y1=sin(5t)*(0<t<25)+sin(t)*(25<t<50)', 'y2=sin(t)*(0<t<25)+sin(5t)*(25<t<50)'两个函数的傅里叶变换，以及其左右两半部分的傅里叶变换。
2、'window.m'程序画出了窗函数的图，有助于理解窗函数的作用。
3、'STFT_main.m'程序计算了'y = sin(20*t).*(t<25) + sin(t).*(t>=25)'函数的在窗函数宽度为10，位置分别为[5,25,45]时的傅里叶变换，还计算了'y = sin(20*t).*(t<25) + sin(t).*(t>=25)'函数的在窗函数宽度为10的短时傅里叶变换。
4、'STFT_funciton.m'为STFT_main的组成函数。
5、'base_function.m'画出了FT和DTFT的基函数的图，有助于从基函数角度理解FT和DTFT。
6、本程序调用了'../Basic'目录下的My_FFT函数进行快速傅里叶变换，你也可以调用Matlab自带的fft函数进行傅里叶变换。
7、由于第二篇文章还涉及到了DFT，你可以在'../Basic'目录下找到My_DFT函数。My_FFT函数是My_DFT函数的快速算法，两个函数的计算结果相同。
